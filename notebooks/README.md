# MOT-MIC Tutorials

These notebooks follow the same tutorial rhythm as the scTour basic inference documentation: dataset, data loading, model training, inference, visualization, validation, and robustness.

| Notebook | Dataset | Purpose | Ground-truth level |
|---|---|---|---|
| `01_motmic_example.ipynb` | Synthetic paired data | Fully runnable quick start | Simulated labels |
| `02_GSE173958_lineage_validation.ipynb` | GSE173958 | Lineage-validated MIC inference | Strongest available |
| `03_GSE249057_timecourse_discovery.ipynb` | GSE249057 | Time-course MIC-like state discovery | Weak labels |
| `04_GSE178318_human_crc_liver_metastasis.ipynb` | GSE178318 | Human CRC primary-to-liver validation | Clinical weak validation |
| `05_GSE277783_spatial_validation.ipynb` | GSE277783 | Spatial MIC program validation | Spatial weak validation |
